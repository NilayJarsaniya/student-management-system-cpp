# Student Management System (C++)

## 📌 Description
A console-based application to manage student records using file handling in C++.

## 🚀 Features
- Add Student
- Display All Students
- Search Student
- Update Student
- Delete Student

## 🛠️ Technologies Used
- C++
- File Handling

## ▶️ How to Run
1. Compile the code using:
   g++ main.cpp -o app
2. Run:
   ./app

## 📂 File Used
- students.dat (stores records)
